# Food Delivery Analytics Project

## 📌 Overview
This project analyses a synthetic food delivery dataset using Python, SQL (DuckDB), and Tableau. It explores revenue trends, customer behaviour, restaurant performance, and cohort retention through interactive dashboards and data visualisations.

## 📊 Dashboards (Tableau)
The Tableau workbook contains four dashboards:

1. **Overview Dashboard**  
   High-level KPIs, daily revenue trends, and monthly activity.

2. **Customer Insights Dashboard**  
   Customer behaviour, reorder patterns, loyalty metrics, and cohort retention.

3. **Restaurant Performance Dashboard**  
   Revenue by restaurant, cuisine performance, and menu item contribution.

4. **Executive Summary Dashboard**  
   A concise business-focused summary of key insights.

Screenshots of all dashboards are included in the `/tableau/screenshots/` folder.

## 🐍 Python Workflow
The Python script (`data_analytics.py`) performs:

- Synthetic dataset creation  
- SQL analysis using DuckDB  
- Daily and monthly revenue analysis  
- Customer behaviour and loyalty metrics  
- Restaurant performance metrics  
- Cohort retention analysis  
- Visualisations using Seaborn and Matplotlib  

This script mirrors the logic used to build the Tableau dashboards.
